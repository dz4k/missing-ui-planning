## Framework-free
No. It is designed for React/Next.js.

## Framework-independent
No.

## Headless
Yes, it is built on Radix UI.
> "Built on top of Radix UI primitives, which are headless and provide logic and accessibility without styling." (https://medium.com/@shadcn-ui-overview)

## Has styles
Yes. It provides pre-styled components using Tailwind CSS.
> "shadcn/ui provides pre-styled components using Tailwind CSS, offering 'beautiful defaults' and carefully chosen default styles out-of-the-box." (https://medium.com/@shadcn-ui-overview)

## Styleable
Fully styleable.
> "Developers own the code and can customize these styles extensively." (https://medium.com/@shadcn-ui-overview)

## Classless
No.

## Buildless
No.
> "It is NOT a component library. It's a collection of re-usable components that you can copy and paste into your apps." (https://ui.shadcn.com/docs)
