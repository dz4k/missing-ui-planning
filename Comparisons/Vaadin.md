# Vaadin Web Components

## Framework-free
Yes.
> "Vaadin components are built as Web Components, making them compatible with any framework or no framework at all." (https://vaadin.com/docs/latest/components)

But: most code examples are for Java, React and Lit, and the API is unergonomic for frameworkless usage
