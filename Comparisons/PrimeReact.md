## Framework-free
No.

## Framework-independent
No.

## Headless
Yes, via Unstyled Mode.
> "When unstyled mode is enabled, PrimeReact components do not include any CSS selectors, and their default styling is removed." (https://primereact.org/unstyled/)

## Has styles
Yes.
> "PrimeReact is a rich set of open source native UI Components for React." (https://primereact.org/)

## Styleable
Config-styleable (Default) / Fully styleable (Unstyled Mode).
> "This approach gives developers complete control over the component's appearance, allowing integration with any CSS framework (like Tailwind CSS)." (https://primereact.org/unstyled/)

## Classless
No.

## Buildless
No.
